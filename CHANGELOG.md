# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-29

### Added
- Initial release of BookMarketing plugin
- Campaign orchestration across multiple channels
- Amazon Ads integration with keyword optimization
- Facebook/Instagram Ads with lookalike audiences
- BookBub Featured Deal applications
- Email marketing automation with sequences
- Social media scheduling and management
- CRM integration with engagement scoring
- NetGalley ARC distribution
- Analytics dashboard with ROI tracking
- PDF report generation

### Integrations
- MageAgent for AI-generated marketing copy
- NexusCRM for reader relationship management
- ProseCreator for book project integration
